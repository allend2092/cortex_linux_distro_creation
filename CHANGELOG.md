# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-03-09

### Added
- Initial pre-alpha release
- Meta-installer for Ubuntu 24.04 systems
- Hardware detection (CPU, GPU, RAM)
- Modular installation profiles (AI/Dev, Security, Finance, General)
- Documentation suite (README, SPEC, META_INSTALLER, ETHICS, CONTRIBUTING)
- First-boot wizard setup

### Known Issues
- Installer still in development (not production-ready)
- Security containers not yet fully tested
- GPU driver installation may fail on some configurations

### Planning
- Full ISO installer (v0.2)
- Automated testing suite (v0.3)
- Official documentation site (v1.0)
